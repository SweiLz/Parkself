วิธีการใช้งาน

clone
```
cd catkin_ws/src/
git clone https://github.com/FIBO-KMUTT/Parkself.git
```

เปิด Terminal [Alt + Ctrl + T]

```
roslaunch parkself_slam parkself_slam.launch
```

เปิดอีก Terminal

```
rosrun parkself_mbed parkself_mbed.py
```
